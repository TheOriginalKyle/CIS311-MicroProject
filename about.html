It all started...

This site was originally created as an extra credit project for my Responsive Web Design Class. And boy was it a doozy. Just check out the requirements.

  • Create a "serverless" website using AWS.
  • Implement a navbar that can be edited in a single file.
  • Utilize Bootstrap throughout.
  • Allow cloud storage for authenticated accounts.
  • Build your own userauth while still being serverless.
  • Add an optional Facebook Login.
  • And many more..

Fortunately I mostly completed the site in time (I had 3 weeks).

Here's a Youtube video of what I ended up turning in.

Current Features

  • Completely Serverless
  • Bootstrap Throughout
  • Cloud Storage
  • Facebook Login
  • All built using AWS (Gotta love that Free Tier)
