SDN Project

The goal of this academic project was to utilize OpenDaylight's API's to automate the port security of a Juniper Switch. In practice this means that when a device connects to a port its MAC address would auto-magically be locked to that port. In addition to a PoE light lighting up to indicate that the port was taken. A very fun project with some unique challenges.
